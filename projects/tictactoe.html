Virtually unwinnable Tic Tac Toe

Introduction

I have created a simple Tic Tac Toe, but the only thing is you have to play with an opponent who is so strong that it is virtually unwinnable. If you want to try this game and challenge your self, I will put a link in the conclusion.

Implementation:

I have used to React JS to make this game. This application uses an AI technique called Minimax Algorithm. The opponent analyses all the possible moves that are ever possible and decide the best move assuming the opponent behaves optimally. This is virtually unwinnable, the best one can do is to draw the game.

Conclusion:

I had a lot of fun developing this application and the power of React is excellent. If you are interested in checking this game click here. Project URL:https://tic-tac-toe-bae90.web.app
