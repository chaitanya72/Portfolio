What’s App Clone developed using React

Introduction

I have developed this project to increase my understanding of React JS. This project taught me various powerful features of React and I have realized the power of React by doing this project. This project is a working clone of what’s app with very less feature set but it is live and deployed using google’s firebase. I will place a link in the conclusion for you to check if you are interested in it. This application has a whats app like UI and people can chat with each other using the rooms. There is no one to one chats in this application. The chats get updated in real-time.

Implementation:

I have used React Hooks, React Context API, and google Firebase Firestore to implement this project. I have used Material UI for icons and deployed the application using Firebase. Firestore is a real-time NoSQL database. This helped to deliver the messages in real-time. The users first need to authenticate them selfs using Google’s Authentication via Gmail.

Conclusion:

I have developed this project using various resources and have realized the power of React JS. If you are interested to check this application, just click here. Project URL: https://whats-app-clone-4fa71.web.app
